# create-workflow

## Objetivo
Criar novo workflow versionado a partir de uma tarefa recorrente.

## Processo
1. Identificar objetivo recorrente.
2. Criar arquivo em .agents/workflows/[nome].md
3. Incluir:
   - objetivo
   - regras
   - entrada esperada
   - processo
   - validação
   - saída obrigatória
4. Atualizar operator-master.md se necessário.
5. Commitar e abrir/atualizar PR.
