# ADR-0001: System Architecture

## Status
APPROVED

## Context
Defining the canonical data sources and execution layers for the Ora Brasiliae operating system.

## Decision
- **GitHub**: Canonical source of truth for code and versioned documentation.
- **Notion**: Capture layer for insights, unstructured data, and high-level roadmaps.
- **Antigravity**: Primary execution agent for coding, automation, and governance tasks.
- **Workflows**: Operational layer defining standardized procedures and guardrails.

## Consequences
- Strict traceability of all changes via Git.
- Separation of concerns between unstructured capture (Notion) and structured execution (GitHub).
- Automated governance through standardized workflows.
