# Tool Roles

## Notion
- Gestão de conhecimento não estruturado.
- Planejamento de alto nível (Roadmap).
- Captura de rascunhos e ideias.

## GitHub
- Fonte canônica de código e documentação.
- Governança via Pull Requests.
- Rastreabilidade de alterações.

## Antigravity
- Agente principal de codificação e execução de tarefas técnicas.
- Responsável por manter a estrutura e integridade do repositório.
- Deve operar somente em branches dedicadas.
- Não pode executar force push, orphan branches, reset hard ou reescrita de histórico sem aprovação humana explícita.

## Ollama/Gemma
- Processamento local de linguagem.
- Classificação de documentos e sumarização.

## Perplexity
- Motor de busca e pesquisa externa.
- Validação de referências bibliográficas e técnicas.

## PowerShell/Python
- Automação de tarefas de sistema.
- Execução de experimentos e scripts de validação.

## GitHub Actions
- Automação de CI/CD.
- Execução de testes e auditorias automatizadas.

## Guto
- Auditor cognitivo e revisor lógico.
- Garantia de integridade epistemológica.
