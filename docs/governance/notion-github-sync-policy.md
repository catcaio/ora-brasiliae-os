# Notion-GitHub Sync Policy

## Direção Notion → GitHub
- Apenas conteúdo com status **APPROVED** pode ser movido para o GitHub.
- A sincronização deve ocorrer via branch dedicada e submissão de Pull Request.
- É proibido o merge automático de conteúdos vindos do Notion.

## Direção GitHub → Notion
- O Notion pode receber atualizações de status, changelogs, links para PRs e SHAs de commit.
- O objetivo é manter a visibilidade gerencial sem comprometer a integridade do código.

## Restrições
- **Proibido sobrescrever GitHub a partir do Notion**: O repositório git é soberano.
- Conflitos devem ser resolvidos manualmente no ambiente git.
